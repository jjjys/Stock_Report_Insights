# Git Commit message rules
1. feat: 새로운 기능 추가
2. fix: 버그 수정
3. docs: 문서/README/주석
4. refactor: 코드 리팩토링 (동작 변화 없음)
5. chore: 빌드, 의존성, 설정 등 기타 작업



# Naver_Securities_Report

1. pip install seleniumbase
2. 코드 제대로 수행 확인
3. pip install -q -U google-genai
