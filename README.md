# Naver_Securities_Report
네이버 증권 리포트 pdf 파일 데이터 수집 후 GEMINI 활용하여 PDF 분석하여 투자 전략 구현


파이썬 3.10 버전으로 다시 진행.
1.pip install seleniumbase
2.코드 제대로 수행 확인
3.pip install -q -U google-genai
4.gemini 공식 문서 확인 및 개발
